## Description

API server

## Installation

```bash
# in the root folder
$ yarn install
$ cp .env_default .env # set your parameters
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# e2e tests
$ yarn run test:e2e
```
